# Настройки Alignment (alignment.yaml)

Этот файл содержит конфигурацию для этапа выравнивания модели (alignment), включая такие методы, как GRPO.

## Параметры

* **`stage`** (строка): Определяет этап пайплайна.
  * *Пример*: `"alignment"`
* **`method`** (строка): Метод выравнивания, который будет использоваться.
  * *Пример*: `"grpo"`
* **`learning_rate`** (число с плавающей точкой): Скорость обучения.
  * *Пример*: `1.0e-5`
* **`batch_size`** (целое число): Размер пакета (batch size) для обучения.
  * *Пример*: `4`
* **`max_prompt_length`** (целое число): Максимальная длина промпта в токенах.
  * *Пример*: `512`
* **`max_completion_length`** (целое число): Максимальная длина генерируемого ответа в токенах.
  * *Пример*: `1024`
* **`epochs`** (целое число): Количество эпох обучения.
  * *Пример*: `1`
* **`num_generations`** (целое число): Количество генераций на каждый промпт (используется в GRPO).
  * *Пример*: `4`
* **`reward_funcs`** (список строк): Список функций вознаграждения, используемых для оценки качества генерации.
  * *Пример*: `["accuracy", "variance"]`
* **`dataset_path`** (строка): Путь к датасету или формат данных.
  * *Пример*: `"json"`

## Пример использования
```yaml
stage: alignment
method: "grpo"
learning_rate: 1.0e-5
batch_size: 4
max_prompt_length: 512
max_completion_length: 1024
epochs: 1
num_generations: 4
reward_funcs:
  - "accuracy"
  - "variance"
dataset_path: "json"
```
