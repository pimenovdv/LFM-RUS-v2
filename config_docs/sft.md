# Настройки SFT (sft.yaml)

Этот файл описывает конфигурацию для этапа supervised fine-tuning (SFT) - обучения с учителем.

## Параметры

* **`stage`** (строка): Определяет этап пайплайна.
  * *Пример*: `"sft"`
* **`batch_size`** (целое число): Размер пакета на устройство во время обучения.
  * *Пример*: `16`
* **`packing`** (булево): Использовать ли упаковку последовательностей (sequence packing). Позволяет объединять несколько коротких примеров в одну последовательность до `max_seq_length` для повышения эффективности обучения.
  * *Пример*: `true`

## Пример использования
```yaml
stage: sft
batch_size: 16
packing: true
```
