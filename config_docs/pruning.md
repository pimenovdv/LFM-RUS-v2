# Настройки Pruning (pruning.yaml)

Этот файл описывает конфигурацию для этапа прунинга (прореживания) модели, который уменьшает размер модели за счет удаления наименее важных весов.

## Параметры

* **`stage`** (строка): Определяет этап пайплайна.
  * *Пример*: `"pruning"`
* **`threshold`** (число с плавающей точкой): Порог прунинга. Значения весов ниже этого порога (или процент удаляемых весов, в зависимости от реализации) обнуляются.
  * *Пример*: `0.05`

## Пример использования
```yaml
stage: pruning
threshold: 0.05
```
