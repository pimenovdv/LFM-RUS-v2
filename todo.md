# План внедрения обучения Diffusion Language Models (MDLM) в пайплайн LFM-RUS-v2

Этот документ содержит пошаговый план разработки для интеграции маскированной дискретной диффузии в существующий пайплайн обучения. Каждый чекбокс соответствует объему работ примерно на один Pull Request (PR).

## [x] Шаги 1-8: Интеграция пайплайнов и базовые методы сэмплирования
**Цель:** Базовая интеграция модели, адаптация пайплайнов CPT/SFT/Alignment, Controllability, а также внедрение базовых методов сэмплирования (Dynamic CFG, Top-k, Top-p, Min-p, Repetition Penalty).

## [x] Шаг 9: Расширенные стратегии сэмплирования (Frequency & Presence Penalty)
**Цель:** Добавить поддержку штрафов за частоту и присутствие токенов в генерацию MDLM.
* **Детали реализации:**
  * Добавить параметры `frequency_penalty` и `presence_penalty` в функцию `generate`.
  * Применять штрафы к логитам на основе контекста перед сэмплированием.
  * Покрыть тестами новые параметры генерации.
