# План внедрения обучения Diffusion Language Models (MDLM) в пайплайн LFM-RUS-v2

Этот документ содержит пошаговый план разработки для интеграции маскированной дискретной диффузии в существующий пайплайн обучения. Каждый чекбокс соответствует объему работ примерно на один Pull Request (PR).

## [x] Шаг 1-7: Базовая интеграция, пайплайны обучения и расширенная генерация
**Цель:** Базовые компоненты (кастомное внимание, конфигурация, модель), адаптация CPT/SFT пайплайнов, Alignment (VRPO, CJ-GRPO), Controllability (CFG, Activation Steering), а также улучшение алгоритмов генерации (Dynamic CFG, Top-k/Top-p фильтрация).

## [x] Шаг 8: Расширенные стратегии сэмплирования (Repetition Penalty и Min-p)
**Цель:** Добавить поддержку новых методов сэмплирования в MDLM.
* **Детали реализации:**
  * Добавить `repetition_penalty` в генерацию.
  * Добавить `min_p` фильтрацию в генерацию.
