# План внедрения обучения Diffusion Language Models (MDLM) в пайплайн LFM-RUS-v2

Этот документ содержит пошаговый план разработки для интеграции маскированной дискретной диффузии в существующий пайплайн обучения. Каждый чекбокс соответствует объему работ примерно на один Pull Request (PR).

## [x] Шаг 1-6: Базовая интеграция, CPT, SFT, Alignment, и Controllability
**Цель:** Создать базовые компоненты (кастомное внимание, конфигурация, модель, генерация), адаптировать CPT, SFT пайплайны, интегрировать алгоритмы Alignment (VRPO, CJ-GRPO), и внедрить доменную специализацию (CFG, Activation Steering).
* **Детали реализации:**
  * Двунаправленное внимание без каузальной маски.
  * Реализованы `DiffusionConfig` и `DiffusionModelForConditionalGeneration`.
  * Генерация с итеративным сэмплированием, unmasking/remasking, CFG.
  * CPT & SFT (WSD, Block Diffusion, Blockwise SFT).
  * Добавлено извлечение и применение векторов управления (Activation Steering).

## [x] Шаг 7: Расширенная генерация и сэмплирование
**Цель:** Улучшить алгоритмы генерации для MDLM.
* **Детали реализации:**
  * Реализовать динамическое планирование CFG (Dynamic CFG Scheduling) - поддержка `linear` и `cosine` расписаний для CFG scale в зависимости от шага.
  * Добавить Top-k и Top-p (nucleus) фильтрацию логитов на каждом шаге деноизинга для повышения качества генерации.
