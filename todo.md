# План внедрения обучения Diffusion Language Models (MDLM) в пайплайн LFM-RUS-v2

Этот документ содержит пошаговый план разработки для интеграции маскированной дискретной диффузии в существующий пайплайн обучения. Каждый чекбокс соответствует объему работ примерно на один Pull Request (PR).

## [x] Шаг 1-5: Базовая интеграция диффузионных моделей, CPT, SFT и Alignment
**Цель:** Создать базовые компоненты (кастомное внимание, конфигурация, модель, генерация), адаптировать CPT и SFT пайплайны, и интегрировать алгоритмы Alignment.
* **Детали реализации:**
  * Реализовано двунаправленное внимание (без каузальной маски).
  * Создана `DiffusionConfig` и `DiffusionModelForConditionalGeneration`.
  * Реализованы механизмы генерации (итеративное сэмплирование, confidence-based unmasking, remasking).
  * Адаптирован CPT: WSD расписание, Top-k Checkpoint Merge, Block Diffusion.
  * Адаптирован SFT: SFT-маскирование и Blockwise SFT.
  * Интегрированы CJ-GRPO/VRPO для Alignment, адаптированы вознаграждения.

## [x] Шаг 6: Доменная специализация (Controllability & Steering)
**Цель:** Внедрить механизмы управления генерацией для специализированных доменных моделей.
* **Детали реализации:**
  * Реализовать дискретный Classifier-Free Guidance (CFG) с нормализацией столбцов матрицы интенсивностей (rate matrix column normalization) для устранения фрагментации текста и обеспечения строгих доменных ограничений.
  * Реализовать механизмы активационного управления (Activation Steering): извлечение векторов направлений (directions) из контрастивных промптов (forward pass).
  * Добавить применение вектора управления (глобальная интервенция) к активациям residual-слоев во время обратной диффузии.
  * Убедиться в корректной поддержке кросс-языкового трансфера (перенос векторов ограничений с английских данных на русскоязычную генерацию).
