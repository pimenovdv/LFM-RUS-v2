# Подготовка данных (Data Prep & Filtering)

## Формат данных
На вход пайплайна DataTrove подаются гигантские сырые дампы (например, срезы Common Crawl, дампы Хабра, Википедии, дампы кода) в форматах:
* `.jsonl`, `.jsonl.gz`, `.jsonl.zst`, `.parquet`.
Каждая запись обычно содержит поля: `text` (содержимое документа), `url` (источник), `metadata` (дополнительная информация, язык и т.д.).

Пайплайн Data Prep включает очистку, фильтрацию (спам, SEO, циклические повторения) с использованием классификаторов и нечеткую дедупликацию (MinHash LSH). На выходе получаются чистые, дедуплицированные датасеты в формате `.parquet` или `.jsonl`, готовые к подаче в этап CPT.

## Требуемое количество данных (Исходный сырой корпус)
Поскольку процесс Data Prep отсеивает значительную часть мусора и дубликатов (иногда до 70-80% исходного объема веб-корпусов), на вход требуется подавать значительно больше данных, чем потребуется для этапа CPT.

Ориентировочные объемы *сырых* данных, которые нужно пропустить через Data Prep, чтобы получить необходимый объем чистых токенов для моделей разных размеров (исходя из соотношения "20-40 токенов на 1 параметр" и коэффициента выживаемости данных после дедупликации ~30%):

* **100m**: Требуется ~2-4 млрд чистых токенов -> На вход Data Prep: **~10-15 млрд сырых токенов** (~30-45 ГБ текста).
* **350m**: Требуется ~7-14 млрд чистых токенов -> На вход Data Prep: **~35-50 млрд сырых токенов** (~100-150 ГБ текста).
* **0.5b**: Требуется ~10-20 млрд чистых токенов -> На вход Data Prep: **~50-70 млрд сырых токенов** (~150-200 ГБ текста).
* **1b**: Требуется ~20-40 млрд чистых токенов -> На вход Data Prep: **~100-130 млрд сырых токенов** (~300-400 ГБ текста).
* **2b**: Требуется ~40-80 млрд чистых токенов -> На вход Data Prep: **~200-260 млрд сырых токенов** (~600-800 ГБ текста).
* **4b**: Требуется ~80-160 млрд чистых токенов -> На вход Data Prep: **~400-500 млрд сырых токенов** (~1.2-1.5 ТБ текста).
* **8b**: Требуется ~160-320 млрд чистых токенов -> На вход Data Prep: **~800-1000 млрд сырых токенов** (~2.5-3 ТБ текста).
